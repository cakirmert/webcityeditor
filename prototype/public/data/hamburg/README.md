# Hamburg Hosted Data

Place small Hamburg CityJSONSeq demo tiles here so GitHub Pages serves them from
the same origin as the app.

Current demo file:

```text
prototype/public/data/hamburg/hamburg-center-alkis.city.jsonl
```

It is generated by `npm run data:hamburg-center` from official Hamburg ALKIS
building footprints exposed through the public ArcGIS FeatureServer. It is not
LoD2: footprints and building attributes are real, while the demo extrusion
height is derived from storey count where present.

The FileLoader reads `prototype/public/data/manifest.json` and only shows hosted
Hamburg quick samples when the referenced file exists. This avoids CORS problems
and avoids broken demo buttons.

Keep this directory for small browser-safe fixtures only. Large citywide exports
belong in GitHub Releases, external object storage, or the future backend.
